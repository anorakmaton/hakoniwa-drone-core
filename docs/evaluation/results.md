
# 主要ドローンシミュレータの機能比較評価

このドキュメントでは、主要なドローンシミュレータの機能を比較評価し、各シミュレータの強みと弱みを明確にします。以下の表は、各シミュレータの対応状況を示しています。

対応状況は以下の記号で示します。
- Ο: 対応されている
- Δ: 対応されているが、制限や注意点あり
- X: 対応されていない

ただし、これらの評価は、各シミュレータのバージョンや設定によって異なる場合があります。最新の情報は公式ドキュメントを参照してください。
