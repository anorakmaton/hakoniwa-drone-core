1. タスクの説明：
icomm_connector.hpp を渡します。
ICommIOのsend APIのAPI仕様書を以下のテンプレートの書式に従って作成してください。

2. 前提情報
send API は、インタフェースです。
実体は、ICommServer もしくは ICommClientのcreate API で生成されたインスタンスとなります。

また、本APIを利用する場合は、必ず、comm_init()を呼び出す必要があることを注意点として記載してください。

出力は、README.md形式でテキストで日本語出力してください。

テンプレートの書式は、api-template.md を参照してください。

3. API仕様の意図：
与えられた引数のデータに対して、create時に指定した通信方式でデータを送信するものです。
引数の異常ケースは、全て、false を返します。

4. APIのプロトタイプ：
virtual bool send(const char* data, int datalen, int* send_datalen) = 0;
